    查询指定用户 【GET】

    查询指定用户 【GET】

    接口信息

    API Path

    /api/rest/v2.0/users/{userId}

    请求协议

    HTTP

    请求方法

    GET

    参数

    请求HEADER

    头部标签 头部内容 说明
    Authorization Bearer {token} {token}从登录接口返回中获取
    Content-Type application/json;charset=UTF-8

    URL路径参数

    参数名 必填 说明
    {userId} 是 要查询的用户的id或者用户名

    URL查询参数

    参数名 必填 说明
    userIdType 否 userId的类型，取值为"username"时表示路径中的userId为用户名。

    URL例子

    /api/rest/v2.0/users/770009

    /api/rest/v2.0/users/test?userIdType=username
