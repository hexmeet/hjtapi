    会议预约API说明

    会议预约API说明

    一、常量

    会议的分屏模式

    分屏模式的值列举如下： auto 1X1，1X2，2X1，2X1H，1X2V， 1TP2H，1BP2H，1LP2V， 2X2，1BP3H，1LP3V，1TP3H， 1LP4V，1BP4H，1TP4H， 1P5，2X3， 1P7， 3X3，1MP8，1BP8，1TP8， 2MP8，1CP12，4X4，1P16，1CP16，4X5，4X5S， 1P20，1CP20，5X5，5X6，5X6S，6X6

    说明： T表示Top，B表示Bottom，C表示Center，M表示Middle，L表示Left，H表示Horizontal，V表示Vertical，P表示Plus，S表示Square

    会议状态

    取值（类型为String） 说明
    APPROVED 预约成功
    ONGOING 正在进行中
    FINISHED 已结束
    FAILED 失败
    DELETED 已被删除

    用户状态

    取值（类型为int） 说明
    0 激活
    1 锁定
    2 停用

    用户角色

    取值（类型为String） 说明
    SYS_ADMIN 系统管理员
    AGENT_USER 客服代表
    ORG_ADMIN 公司管理员
    DEPT_ADMIN 部门管理员
    USER 普通用户

    终端状态

    取值（类型为String） 说明
    OFFLINE 离线
    ONLINE 在线
    INCALL 呼叫中
    UNKNOWN 未知

    终端类型

    取值（类型为String） 说明
    M16 -
    M18 -
    VE/VL -
    HEXMEET 软终端
    OTHER 其他类型

    呼叫协议

    取值（类型为String） 说明
    SVC -
    SIP -
    H.323 -

    双流分屏

    取值（类型为String） 说明
    oneByOne 平衡
    contentOnly 仅双流
    largeContentSmallPeople 双流优先
    largePeopleSmallContent people流优先
    peopleOnly 仅people流

    录制/直播类型

    取值（类型为String） 说明
    GW_WEBCAST 高清
    GW_FHDWEBCAST 超清

    录制状态

    取值（类型为String） 说明
    started 录制中
    paused 已暂停录制
    resumed 已恢复录制
    stopped 已停止录制

    直播状态

    取值（类型为String） 说明
    started 直播中
    stopped 已停止直播

    文件类型

    取值（类型为String） 说明
    AVATAR 用户头像
    ORG_LOGO 公司LOGO
    CONF_RECORD 会议的录制文件
    WHITEBOARD_FILE 会议的白板文件

    文件链接的用途

    取值（类型为String） 说明
    PLAYBACK 播放
    DOWNLOAD 下载
    COPY_URL 复制链接

    会议室的容量

    即会议室能同时容许多少终端入会，容量值列举如下：

    10, 25, 50, 100, 200, 400, 1000.

    二、消息结构体定义

    RestConfReq

    简要描述：

    • 预约会议请求体

    参数说明:

    字段 类型 必选 默认 说明
    name String 是 - 会议名称,长度不可超过128个字节
    startTime long 是 - 会议的开始时刻（以毫秒为单位的Unix时间戳），0表示该会议立即开始。
    duration long 是 - 会议的持续时长（单位为毫秒）,建议值为3600000，即一个小时
    lecturerEpId long 否 - 被设置为主会场的某个终端的id
    lecturerUserId long 否 - 被设置为主会场的某个用户的id
    layout String 是 - 取值见常量：会议的分屏模式
    profile String 是 - 会议质量，"SVC"表示质量优先，"SVC_ECO"表示带宽优先，建议值为"SVC".
    password String 否 - 会议密码
    description String 否 - 会议备注
    endpointIds List(Long) 否 - 与会终端id的列表
    userIds List(Long) 否 - 与会用户id的列表
    roomId long 否 - 指定的云会议室的id（适用于云会议室模式）
    maxPartyCount int 是 - 最大入会方数
    useRandomNumericId boolean 是 false 使用随机号码作为会议号码，端口模式下必须设为true
    redialingEnabled boolean 否 false 自动重拨
    recordingEnabled boolean 否 false 录制
    recordingProfile String 否 - 取值见常量：录制/直播类型,建议值为"GW_WEBCAST".
    recordingLayout String 是 - 取值见常量：双流分屏，建议值为"oneByOne".
    liveStreamingUrl String 否 - 推流地址
    adminUserId long 否 - 备选会控者的id
    subtitle RestSubtitle 是 - 字幕
    autoMutingEnabled boolean 否 false 开启自动静音
    allowAnonymousParty boolean 否 false 允许匿名用户入会

    RestSubtitle

    简要描述：

    • 会议字幕

    参数说明:

    字段 类型 说明 必选 备注
    on boolean 启用字幕 是 -
    content String 字幕内容 是 -
    displaySpeed int 滚动速度 是 建议取值：0表示静止，2表示慢速，5表示中速，8表示快速。
    displayRepetitions int 滚动次数 是 滚动速度大于0时才生效.最小值为1,最大值为20,建议值为20.
    verticalBorder int 垂直位置 是 最小值为0，最大值为100，建议值为50.
    transparency int 透明度 是 最小值为0，最大值为100，建议值为50.
    fontSize int 字体大小 是 最小值为9，最大值为32，建议值为24.
    foregroundColor String 前景色 是 十六进制颜色码，建议值为"#FFFF00".
    backgroundColor String 背景色 是 十六进制颜色码，建议值为"#3300CC".

    简要描述：

    • 会议返回体

    参数说明:

    字段 类型 说明
    id long 会议的ID,后续调用会议控制api时用此id来指明要控制哪个会议

    startTime long 会议的开始时刻（以毫秒为单位的Unix时间戳）

    guid String -
    status String 取值见页面：会议状态
    numericId String 会议号
    hostZoneAddress String 会议的主可用区的地址
    applicant RestUser 会议的申请人
    adminUser RestUser 会议的备选会控者
    lecturerEpId long 被设置为主会场的某个终端的id
    lecturerEpName String 被设置为主会场的某个终端的名称
    lecturerUserId long 被设置为主会场的某个用户的id
    layout String 取值见页面：会议的分屏模式
    profile String 会议质量，"SVC"表示质量优先，"SVC_ECO"表示带宽优先.
    password String 会议密码
    description String 会议备注
    endpoints List<RestEndpoint> 与会终端的列表
    users List<RestUser> 与会用户的列表

    isRandomNumericId boolean 使用随机号码作为会议号码
    redialingEnabled boolean 自动重拨
    recordingEnabled boolean 录制

    recordingLayout String 双流分屏，"oneByOne"表示平衡，"contentOnly"表示仅双流，"largeContentSmallPeople"表示双流优先.
    liveStreamingUrl String 推流地址
    subtitle RestSubtitle 字幕
    inCallPartyCount int 当前已成功连接到会议的与会者的数量
    version int 此条会议数据的版本号
    autoMutingEnabled boolean 开启自动静音
    allowAnonymousParty boolean 允许匿名用户入会

    三、会议预约API说明

    (1) 预约会议

    简要描述：

    • 用于预约会议

    请求URL：

    • /api/rest/v2.0/conferences

    请求方式：

    • POST

    请求参数：

    参数名称 是否必选 详细描述
    token 是 登录令牌

    请求Header:

    Content-Type: application/json;charset=UTF-8

    请求体说明： RestConfReq

    成功返回体说明： RestConf

    (2) 获取会议列表

    简要描述：

    • 用于查询会议列表

    请求URL：

    • /api/rest/v2.0/conferences

    请求方式：

    • GET

    请求参数：

    参数名称 是否必选 说明 备注
    token 是 登录令牌 -
    startTime 是 指定查询时间段的起始点 以毫秒为单位的Unix时间戳；若会议的时间段与指定的查询时间段有重叠，则该会议会出现在查询结果里。
    endTime 是 指定查询时间段的结束点 以毫秒为单位的Unix时间戳
    status 否 会议状态 可取值为APPROVED, ONGOING, FINISHED, 参见页面：会议状态
    若不设值，则默认返回状态为APPROVED或ONGOING的会议。
    confNameSlice 否 按名称或会议号查询会议的关键字 -
    deptId 否 会议申请者所在部门的ID -
    includeHierarchicalDepts 否 是否包含下属部门 值为”true”或”false”
    applicantId 否 作为会议申请者的用户的ID -
    maxResults 否 查询结果中会议数量的上限 用来控制查询结果的数量

    成功返回体说明： List<RestConfSummary>
